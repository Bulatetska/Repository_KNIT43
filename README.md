# Repository_KNIT43

Hospital Management System — це комплексна інформаційна система для клініки, побудована на платформі .NET 8.0 з тришаровою архітектурою: ядро бізнес-логіки (Core), демо-консоль (Console) та повноцінний desktop-інтерфейс (WPF, MVVM). Застосунок забезпечує облік пацієнтів, лікарів, відділень, розкладу прийомів, медичних призначень і записів, підтримує ієрархію класів із чистим розділенням відповідальностей, інтерфейси сервісів і можливість одночасної роботи консольного й графічного клієнта. Проєкт реалізовано командою з п’яти студентів через індивідуальні гілки, pull-request-рев’ю та автоматичну збірку, що доводить його готовність до подальшого масштабування й інтеграції з веб-API чи хмарними сервісами.
